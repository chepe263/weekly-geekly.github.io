AeroFS - Synchronize files without a server



How it works




1. Combines p2p technology with modern data replication algorithms to provide the user with a private and distributed file system using his own equipment.
')
2. P2P layer allows you to connect two or more computers, or as much as the network or firewall settings allow. This means that AeroFS will work on the Internet, on the local network and in your office.

3. Since AeroFS is a fully distributed system, if the AeroFS server is down then everything will work for the user.

Cloud or not Cloud? You decide.



We believe end users should have freedom of choice when it comes to managing personal data. AeroFS allows you to optionally add online storage as your personal cloud. In general, being able to store information on the Internet gives you confidence in the safe storage of your important files.

Security



Each AeroFS device has its own 1024-bit RSA key, which is certified by us. When two devices communicate with each other, they perform mutual authentication and establish a secure connection using keys and certificates. Since the connection is encrypted directly from one device to another, data goes from end to end in an encrypted form. As a result, your data is protected, even if this P2P is transmitted from a third party.
