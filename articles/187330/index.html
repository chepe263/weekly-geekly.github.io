Hacked official forum Ubuntu

The news is far from pleasant, but on July 7, 13, the official Ubuntu forum was hacked - ubuntuforums.org
Now is at the stage of recovery.

Now when visiting the forum the following information is given


')
Details of the attack are not yet known, but there is reason to believe that all user information has been compromised, including passwords, login and email address.

However, although the password was stored in a hashed form, it is strongly recommended to change the password from ubuntuforums.org on all services where it was also used.

It is also noted that hacking only concerns the forum and does not affect Ubuntu One with Launchpad or any other Ubuntu / Canonical services.

Among the assumptions that happened, there is a possible use of the old version of the forum engine vBulletin, the vulnerability in which could have caused the hacking.
