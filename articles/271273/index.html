Introducing the new version of Kerio Connect 9



A new version of Kerio Connect 9.0 was released yesterday, with additional time-saving features for end users and administrators. This release also includes the new Kerio Connect Client for improved usability and overall performance.

  • Improvements in Kerio Connect Client - simplified design and increased the speed of processing tasks, such as deleting or moving multiple emails.
  • Filtering, routing, and relaying server rules are server-level rules that allow a variety of actions to be performed with emails, including forwarding, automatic response, and custom notifications.
  • Remote server update - you can remotely update the server from the Kerio Connect admin panel in any browser.
