Software Engineering and Miyamotopoliya course



The Computer Science Student video project continues. Today is a brief description of the Software Engineering course and some videos from the diaries of the Miyamotopoliya game project. The project is the main part of the course, the release and the main run was completed today. In the " Screenshots " section you can also look at various random screenshots of my screen for the last 3-4 days.

I was engaged in the game GUI, that is, the graphic component. It was written in Java, is also a container containing some game states. Contains an xml parser that parses messages about updating all information on the board from the server. A more detailed description of the project with full documentation (~ 60 pages), full source code of the server, client, GUI of the game and admin GUI, and UML models of the system with descriptions of protocols, sequence diagrams, etc. will be available on new days along with a new video.
