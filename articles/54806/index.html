What is Open Storage?



Open architecture

Industry standards in open-source hardware and software with integrated storage system functionality (higher scalability and lower cost of ownership compared to proprietary systems)

Open source software

Open source and API (support for third-party software developers)

Interoperability

Simple and predictable integration of systems in a heterogeneous environment (open standards)

Solid State Drives (SSD) Solid State Drives


Components of SSD:
A) Controller
B) DRAM
C) FLASH bank

Each FLASH chip is part of a shared virtual pool and its space is addressed by the built-in controller.
The controller tracks:
  • Wear leveling
  • CRC
  • Bad block mapping

The controller provides support for interfaces such as SATA, PATA, SAS or FC
