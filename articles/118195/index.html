Sony admitted that the personal data of PlayStation Network users were stolen

Sony admitted that the following user data was stolen from the PlayStation Network database from April 17 to 19:
  • Name
  • Postal address (with zip code)
  • Country of Residence
  • E-mail address
  • Date of Birth
  • PlayStation Network / Qriocity login and password
  • Handle / PSN online ID

It may also have been stolen information about purchases, secret questions (and answers) to access the PlayStation Network / Qriocity.
While there is no confirmation, it is not excluded that the number and end date of the credit cards used in the system (except for the secret code) also fell into the hands of the intruders.
