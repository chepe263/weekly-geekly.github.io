Using Latex in Open Office

If you very often have to use formulas when typing a document, then this post is for you.

Many use MS Word and Equation for this purpose - this is not our way.

All over the world, TeX is used for typing formulas (LaTeX, MikTeX, etc.). But for typing in TeX completely, certain skills are required. However, for OpenOffice there is one remarkable extension - OOoLatex. This is a set of macros to support LaTeX in OO. The extension allows you to use all the features LaTeX, without leaving a convenient editor environment.
