A new version of opensource firewall pfSense 2.1.2 has been released. Less than a week



On April 10, 2014, the distribution release for creating firewalls and pfSense 2.1.2 network gateways was introduced. The distribution is based on the FreeBSD 8.3 codebase, using the m0n0wall project and actively using pf and ALTQ. There are many images available for download for the i386 and amd64 architecture, ranging in size from 80 to 180 MB, including a LiveCD and images for uploading directly to Compact Flash's of various sizes (512, 1GB, 2GB, 4GB).

This release was released less than a week after the announcement of version 2.1.1 . All because of the acclaimed vulnerability OpenSSL "Heartbleed"
