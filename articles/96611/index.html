MODx Evolution 1.0.4

On June 7, a new version of the wonderful content management system MODx Evolution 1.0.4 was released. As the developers say, no new features and capabilities were introduced with this version, however, many bugs were fixed, including at the security level. As always, developers advise to upgrade.



And here, in fact, a few facts that relate to this release:
  • About 50 updates, which include bug fixes, security level updates;
  • Updated additions and translations;
  • Preview now uses friendly URLs if they are included.
