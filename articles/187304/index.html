[Poll] Application for saving and synchronizing history in the clipboard. To be or not to be?

Good day,

I often find myself thinking that it would be nice to find out what was contained in the clipboard yesterday, a week or a month ago ... Or quickly open a link in the browser that is stored in the clipboard on another computer ...

To solve these problems I decided to write a small application. The minimum set of functionality that I plan to create in the first version includes the ability to save the history of the contents of the clipboard, search, synchronize the history with the server and other computers (copied to the clipboard on machine A is automatically copied to the clipboard on machine B)
