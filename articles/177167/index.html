In the new Ubuntu Touch preview builds, the first applications will soon appear.



Despite the fact that the mobile operating system Ubuntu Touch has been presented for a long time, for the time being it is simply impossible to fully work with preliminary versions of this OS. The fact is that it is almost a bare shell, without applications and without the possibility of installing such. But very soon (maybe within a few days), applications, both basic and additional, will finally appear in the Ubuntu Touch preview.

Here are some screenshots of applications for Ubuntu Touch, and these applications will soon appear in the OS. In addition to weather and time-line applications, a file manager, calendar, and clock are being prepared for release. Independent developers are preparing a Twitter application, an application for Reddit, plus display applications (in particular, for XBMC).
')


Michael Hall has already said that the developers will soon see their own applications in the daily cdimage.ubuntu.com/ubuntu-touch-preview/daily-preinstalled (or, more precisely, nightly, if you can put it that way) distributions of Ubuntu Touch preview. Third-party developers can add their own applications to the Ubuntu Touch App Collection , from which, if they are approved, the applications will also be received in an updated distribution.



In other words, Ubuntu Touch is gradually turning from a purely demonstration shell into a full-fledged mobile operating system, which it should be. It is worth recalling that a full-fledged Ubuntu Touch should be released in late 2013 - early 2014.
