Microsoft Search Server 2010 Express

Many companies have file servers where a bunch of user information is stored, which is not very necessary. To search for such file dumps (and not only) there is an excellent product Microsoft Search Server .



At the moment, the current version of Microsoft Search Server 2010. It is based on Sharepoint 2010.
There are two editions complete and express. Editorial comparisons .
')
Microsoft Search Server allows you to search for information of interest to you in the corporate network.
- Supports 31 formats: Microsoft Office formats, HTML, SharePoint sites, Open Document formats, PDF and others.
- shows in the issuance of only those documents where the seeker has access
- searches with different endings (for Russian it is important)
- you can specify the search for a specific directory, file type
- embedded in Windows Explorer (Explorer)
- there is a mobile version
- allows for the analysis of search queries
and much more
