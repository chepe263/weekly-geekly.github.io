Eclipse and SFTP without problems

For a long time, Aptana and Eclipse had a problem that caused a lot of trouble, namely the impossibility of normal operation with a remote machine that ssh has access to. Aptana has full sftp support only in the pro version, and the officially not supported sftp plugin has a lot of problems (the most significant - when the network crashes and raises the plug-in continues to work as if nothing had happened, but does not save files) not updated for at least a year.

Fortunately, the solution exists and the name of it is Remote System Explorer . RSE not only solves the problem of remote file access, but also provides a remote terminal, file search and even remote debugging.
