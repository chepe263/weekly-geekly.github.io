Quake III Server on Nokia Phone

Programmer Olli Hinkka, who has already become relatively famous for porting Quake to Symbian mobile phones, recently released the release of Mobile Quake III Arena . The game works only on phones with the Texas OMAP 2420 chipset, which automatically narrows the range of suitable phones for launching Quake III to Nokia N95 8 GB, N82 and E90.


The times when Quake III required high end iron had passed. Now anyone (if he is the owner of the above Nokia phones) can easily run into one of the most popular of the network shooters. And not only with bots. The fact is that now the user can run a server on his phone. To which other players can connect. Not only that, the Quake III port for Symbian supports connecting Bluetooth keys.

The screenshot, by the way, was made with the Nokia N95 8 GB.
