Android applications can be run on iPad



Now, Android applications can be run on the iPad and on many other non-Android devices.
This is possible with the special application Alien Dalvik 2.0, developed by the Swiss company Myriad. In fact, it is a virtual Java machine in which it will be possible to execute an APK application without being tied to the real Android OS. The goal of developers is not selling their product to end users, but providing it to manufacturers of various devices to support Android applications on them.
