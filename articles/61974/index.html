New version of Linux kernel 2.6.30

A couple of days ago, a new version of the Linux 2.6.30 kernel was released.

From the new:
  • support of the file system of sequential recording of large amounts of information, NILFS2 (log-structured filesystem)
  • POHMELFS (Parallel Optimized Host Message Exchange Layered File System) high performance network distributed file system support
  • file system support for devices with object-oriented data storage EXOFS
  • support for caching file system FS-Cache
  • interprocess communication protocol support for servers in an RDS cluster (Reliable Datagram Sockets)
  • Support for fast kernel loading (Fastboot) from Intel due to asynchronous device polling
  • support of the current specification of the developed standard IEEE 802.11w
