Symfony 1.2

Approximately 9 hours ago version 1.2 of a wonderful symfony framework was released.

What changes have occurred in the version 1.1 framework? There are a lot of changes, so I will focus on the most remarkable ones.

  • Propel 1.3 is used (which means PDO, instead of Creole)
  • Doctrine 1.0 is present in the default delivery. Declared normal support for this ORM, including in the admin generator
  • The admin generator is prettier and more comfortable !!! (probably the most important feature of this release, from my point of view)
  • Many improvements to the routing subsystem (now called the routing framework)
  • Normal operation of nested forms in the form framework
  • Filters for records in tables
  • The debug panel can now be configured
