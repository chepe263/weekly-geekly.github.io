The Japanese have released a browser with three engines


Many developers and ordinary users of the Internet are faced with a situation where they have to use several browsers at once.

For some web applications and access to corporate email, you often have to use standard Internet Explorer, for comfortable work using many extensions - the Firefox browser, and for the usual quick web browsing - the Google Chrome browser. For such demanding users, developers from Lunascape created a browser of the same name that combines the three most popular technologies for processing and outputting web pages - Trident (Internet Explorer), Gecko (Mozilla Firefox) and WebKit (Safari, Google Chrome).

When you open a new tab in the Lunascape browser, you can specify which rendering engine should display it. What is even more convenient - you can clearly determine with the help of a mechanism to open each specific site. To compare the look and behavior of pages in different rendering engines, you can give the command to reload the page in the same tab using a different mechanism.
')
The developers of the Lunascape browser claim that due to the optimized implementation of the Gecko technology, their browser is faster than everyone else. However, the current version of Lunascape 5 Alpha runs rather slowly, and some roughness in the interface still slows down the effective work with the browser.

Browser Lunascape 5 is clearly aimed at enthusiasts of technical progress - it is equipped with all the tools that you can imagine for a web browser. In particular, this version provides support for RSS feeds, including podcasts, saving inputted data in forms, as well as the management of special mouse movements (gestures). In addition, in Lunascape 5, through the menu, you can access the deepest settings for each of the rendering engines used.

Lunascape Browser supports its own extension modules and layouts, as well as add-ons for IE. At the same time, Firefox extensions are not supported - this is a serious drawback. The address bar Lunascape is made without the excesses that favorably distinguish Firefox and Chrome. In general, this browser in the current can be recommended unless the most curious researchers of technology, although the ideas embodied in it, definitely deserve development.
