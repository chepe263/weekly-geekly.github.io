New version of Intel SDK for OpenCL applications

A new version of the free Intel SDK for OpenCL applications - 2013 R2 has been released . We give a list of innovations.
  • Support for Windows 8.1 operating system;
  • Support for new Intel Bay Trail processors with Intel HD Graphics, namely the Intel Atom Z3000 series, Intel Celeron J1000 / N2000, Intel Pentium J2000 / N3000;
  • Trace OpenCL cores on Intel integrated graphics using Intel VTune Analyzer XE 2013 ;
  • New OpenCL extensions cl_accelerator_intel , cl_intel_motion_estimation with support for hardware accelerated embedded motion estimation kernels.

Currently, Windows 8.1 is supported only with the following drivers: Intel Iris driver and HD graphics driver for Windows 8.1 (version 15.33.5) , Intel HD Graphics driver for Windows 8.1 (version 15.33.3) for Intel Bay Trail processors.
Users of Windows 7 and 8 should use the previous release.
