Microsoft recreated 1994 website

Succumbing to the feeling of nostalgia, Microsoft employees picked up a copy of the 1994 first corporate website .



In 1994 there were only a few thousand sites on the Internet. HTML was considered a technical novelty, and the very concept of publishing information on the web was unusual. Virtually no one understood HTML and there were no tools for web development. Many browsers did not support the display of images.

That year, Microsoft opened a website on the Internet, which for those times should look very futuristic: the sunrise on a gray background surrounded by stars and links to reference subsections.
