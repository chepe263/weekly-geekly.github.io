Linux for web developer

Hello dear users and Linux users. I broke down before choosing the OS and decided to install Linux, only in the choice of the distribution, I stumbled.

The fact is that when working on Windows, I have already got used to using various software (fries or trial). And when switching to Linux, I want to see alternatives.

I tried to use Ubuntu, but I immediately saw problems with the fonts on the sites, they were clumsily displayed and would like to know from the pros how to fix the situation.
')
In general, the question is simple: What do you think is the best Linux distribution, suitable for a typical web programmer (Java, PHP, MySQL PostgreSQL, SFTP / SSH-ftp, SSL-FTP support).

Just Windows openly got its lags (even on a machine like the Athlon X2 4800+, 2GB RAM). I want productive work, so that nothing distracts. Therefore, I ask the opinion of professionals. Fortunately, on this site, there are. :)

ps At present I use Nu PHPed (there is support for SSH access, it works more stable than in Zend, you can click right in the code and open it with separate files, search for the declaration of functions, classes and variables, WinSCP as an ftp client, Pts convenient , SQL Manager for working with MySQL DB (pgAdmin for working with PostgreSQL).
