One line of HTML code can delete data or reload Samsung phones

At the security conference, it was demonstrated how you can remotely delete data from the Samsung Galaxy S III in a single line of HTML code. Worst of all: it seems to work on many Samsung smartphones with TouchWiz


This problem is specific specifically for Samsung phones, with TouchWiz installed, and this is a large part of the Samsung line. Phones released without the TouchWiz shell are not subject to potential danger (for example, the Galaxy Nexus exploit does not work).

At the conference, it was demonstrated how the link received on the phone was specially processed using TouchWiz, as a result of which data was deleted from the phone. This exploit as a string of HTML code can also be called via the built-in QR code scanner or NFC.
