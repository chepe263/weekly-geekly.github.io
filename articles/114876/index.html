Release Xen Cloud Platform 1.0

What happened so long talked about in the xen-api mailing list - version 1.0 of the Xen Cloud Platform was released.

Key changes:
  • Upgrading the kernel version (zen is always a very specific topic)
  • Blktap2 support
  • Bexar support (more precisely, on the contrary, Bexar supports XCP)
  • RDAC mulipath support at XAPI level
  • Local network storage caching
  • Managed Auto Backups of Machines
  • Boot hosts from network storage (required HBA required)

From myself I will add (that I noticed new when picking with RC-versions):
  • Thin provisioning on LVM (including iscsiolvm)
  • Support for pure virtual networks between hosts (traffic routing through GRE encapsulation)
  • xapi has finally become installed as RPM packages, and not as a blob-code trash into / opt.
  • Substantial processing of documentation

Xen's blog entry: blog.xen.org/index.php/2011/03/03/xcp-1-0-released
Link to download page: www.xen.org/download/xcp/index_1.0.0.html
