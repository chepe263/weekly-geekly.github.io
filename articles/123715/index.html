useful utility vmware boomerang

Colleagues,
Yesterday, VMware Labs released the first version of a small (7 MB), but useful utility Boomerang .
in fact, this is a very simple client for connecting to virtual machine consoles running ESX (i) servers, and it supports simultaneous connections to different vCenter servers.
In addition, it supports the saving of usernames / passwords for quick connection.
can be useful to anyone who often needs to connect to multiple virtual machines
but there is no connectivity via RDP / VNC, etc.
